# E-commers-web-site-HTML-CSS-Javascript-
Maison is a modern PHP & MySQL eCommerce storefront built with a minimal and responsive UI.
The platform allows businesses to showcase products, display customer reviews, and receive orders directly through WhatsApp.

Features

Modern homepage with hero section

Featured product display

Product categories

Customer review/testimonial system

WhatsApp ordering integration

Contact form

Image gallery

Admin panel for managing products and reviews

Responsive design

Tech Stack

PHP

MySQL

HTML5

CSS3

JavaScript

Use Case

Perfect for small online stores or portfolio eCommerce projects that want a simple storefront with direct customer communication through WhatsApp.
